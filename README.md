📌 Project: Trip Advisor Website Clone

🧑‍💻 This project is a responsive Trip Advisor landing page clone built using HTML, CSS, and JavaScript.
It recreates the layout of the TripAdvisor homepage including navigation, search section, experience cards, and travel destinations.

✨ Features

• 🌍 Navigation bar with Tripadvisor branding and menu options

• 🔎 Search section to explore places, hotels, restaurants, and activities

• 🏨 Interactive category tabs (Hotels, Things to Do, Restaurants)

• 🖼 Image cards showcasing travel experiences and destinations

• ❤️ Favorite icon UI for travel listings

• 🎞 Image slideshow animation in featured section

• 📱 Responsive design for desktop and mobile screens

• 🎨 Modern UI with hover effects and clean layout

🛠 Technologies Used

• HTML5 – Structure of the webpage

• CSS3 – Styling, layout, animations, and responsive design

• JavaScript – Interactive UI updates (category switching)

• Font Awesome – Icons for UI elements

• Google Fonts – Typography for better design

⚙️ How it works

1️⃣ User lands on the homepage

2️⃣ Navigation bar provides access to travel categories

3️⃣ Category buttons update the search heading dynamically

4️⃣ Users can explore travel experiences and destinations

5️⃣ Responsive layout adapts to different screen sizes

🎯 Purpose of the Project

This project was built to practice frontend web development concepts, including:

• Responsive layouts using CSS

• UI design similar to real-world websites

• JavaScript DOM interaction

• Image layout and card-based design

• Building modern landing pages
